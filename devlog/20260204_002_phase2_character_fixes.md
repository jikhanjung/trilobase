## 기술 노트

### 깨진 문자 원인
PDF에서 텍스트 추출 시 특수 문자 인코딩 문제 발생:
- `Ř` → `\x02` (STX 제어문자)
- `Š` → `.` (마침표)
- `ě` → `˛` (ogonek)
- `Č` → `!` (느낌표)
- `ř` → `"` (따옴표)

### 사용된 명령어
```bash
# 제어문자가 포함된 패턴 수정
sed -i $'s/P\x02IBYL/PŘIBYL/g' file.txt

# 일반 패턴 수정
sed -i 's/\.arka Fm/Šárka Fm/g' file.txt
```
