# 114: Diff Tree — 단일 병합 트리 with 변경 색상 코딩

**날짜:** 2026-03-07
**Related:** R02 Phase 2 (Diff Tree), devlog 110 (Profile Diff Table)

## 개요

두 classification profile 간 차이를 단일 트리에 시각화하는 Diff Tree 뷰 구현.
base profile 구조 위에 moved 노드를 compare profile 부모로 재배치하여,
변경 전후를 한 눈에 파악 가능.

## SQL 쿼리 (`profile_diff_edges`)

- Base profile 엣지를 기준으로 compare profile과 LEFT JOIN
- diff_status: `same` / `moved` / `added` / `removed`
- `parent_id_a` (base), `parent_id_b` (compare) 양쪽 부모 ID 반환
- Moved 노드는 JS에서 `parent_id_b`로 재배치 (compare profile 위치에 표시)

## tree_chart.js 변경

| 기능 | 설명 |
|------|------|
| `buildHierarchy()` | diff_mode 감지, diffStatusMap 구축, moved 노드 re-parent |
| `assignColors()` | diff status별 색상 (grey/orange/green/red) |
| `drawLinks()` | diff 색상 링크 (same=faint, moved/added/removed=bold) |
| `drawDiffLegend()` | 우측 하단 범례 — status별 카운트 표시 |
| Tooltip | moved 노드에 `BaseParent → CompareParent` 이동 경로 표시 |

## assertion DB 변경

| 항목 | 내용 |
|------|------|
| `profile_diff_edges` 쿼리 | 두 프로필 엣지 UNION + diff_status + parent_id_a/b |
| `diff_tree` 뷰 선언 | compare_view, tree_chart display, diff_mode 옵션 |
| Ghost edges | 비활성 (show_ghost_edges: false) |
