# UI 개선: Statistics 중복 제거 및 Children 네비게이션

**날짜:** 2026-02-06
**Phase:** Post-Phase 12 (UI 개선)

## 변경 사항

### 1. Statistics 중복 표시 수정

**문제:** Family 이상 rank의 상세정보 모달에서 Statistics 섹션에 "Genera: 185"와 "Genus: 185"가 동시에 표시되는 중복 문제.

- `genera_count` 필드 → "Genera: 185"
- `children_counts` 루프 → "Genus: 185"

**수정:** `children_counts` 렌더링 시 `genera_count`가 이미 표시된 경우 "Genus" 항목을 건너뛰도록 처리.

### 2. Children 목록 클릭 네비게이션

**문제:** rank 상세정보 모달의 Children 목록이 단순 텍스트로만 표시되어 클릭할 수 없었음.

**수정:** Children 항목을 클릭하면 해당 taxon으로 네비게이션:

| Children rank | 동작 |
|---|---|
| Order/Suborder/Superfamily | 트리에서 노드 펼침 + 선택 → 상세정보 모달 |
| Family | 트리 펼침 + 선택 + genus 목록 로드 → 상세정보 모달 |
| Genus | 부모 Family 선택 + genus 목록 로드 → genus 상세정보 모달 |

**추가 함수:**
- `expandTreeToNode(nodeId)` — 트리에서 특정 노드까지의 경로를 펼치고 하이라이트
- `navigateToRank(rankId, rankName, rankType)` — 상위 rank로 네비게이션
- `navigateToGenus(genusId, familyId, familyName)` — Genus로 네비게이션

## 수정 파일

- `static/css/style.css` — clickable children 스타일 추가
