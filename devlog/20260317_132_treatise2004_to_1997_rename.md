# 132: Treatise 2004 → 1997 연도 수정 + PDF_SOURCE_STATUS 갱신

**날짜:** 2026-03-17

## 배경

Treatise Part O Revised (Agnostida + Redlichiida)의 발행연도가 프로젝트 전체에서 2004년으로 잘못 기록되어 있었음. 실제 발행연도는 **1997년** (Kaesler, R.L. (Ed.), 1997, Treatise on Invertebrate Paleontology, Part O, Revised, Vol. 1).

## 작업 내용

### 1. Treatise 2004 → 1997 리네이밍

프로파일명, 소스 파일명, 변수명, reference 연도를 일괄 수정.

**소스 파일 rename:**
- `data/sources/treatise_2004_ch4.txt` → `treatise_1997_ch4.txt`
- `data/sources/treatise_2004_ch5.txt` → `treatise_1997_ch5.txt`
- 헤더 내 reference 연도: 2004 → 1997

**빌드 스크립트 (`scripts/build_trilobase_db.py`):**
- 변수명: `TREATISE_2004_CH4_REF_ID` → `TREATISE_1997_CH4_REF_ID` 등
- 프로파일명: `treatise2004` → `treatise1997`
- reference INSERT 연도: 2004 → 1997
- provenance/schema 연도: 2004 → 1997
- 함수 파라미터: `t2004_ch4_edges` → `t1997_ch4_edges` 등

**변환 스크립트 (`scripts/convert_to_source_format.py`):**
- 함수명: `convert_treatise_2004_ch4/5()` → `convert_treatise_1997_ch4/5()`
- 헤더/출력 파일명 수정

**문서 수정 (활성 파일만):**

- `docs/PDF_SOURCE_STATUS.md` — 소스 파일명

- `.claude/skills/pdf-to-taxonomy/SKILL.md` — 소스 파일 참조

**변경하지 않은 파일:**
