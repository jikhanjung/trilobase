# 109: Treatise 1959 taxonomy 추출 및 assertion DB import

**날짜:** 2026-03-07
**DB:** trilobase-assertion-0.1.4.db

## 개요

1959년 Treatise on Invertebrate Paleontology Part O (Trilobitomorpha) PDF에서 삼엽충 전체 분류 체계를 추출하고, assertion DB에 `treatise1959` 프로필로 import했다.

## 작업 단계

### 1. Higher Taxonomy 수동 입력 (`scripts/extract_treatise1959.py`)

- pp. O160-O167 (Outline of Classification) 이미지를 보고 수동 입력
- 8 orders, 13 suborders, 33 superfamilies, 142 families, 128 subfamilies
- 각 taxon의 author, year, genera_count, range 포함
- 출력: `data/treatise_1959_taxonomy.json` (tree 구조)

### 2. OCR Genus 추출 (`scripts/ocr_treatise1959_genera.py`)

- pp. O172-O525 (PDF pages 191-545) → Tesseract OCR (300 DPI)
- 정규식 패턴: `(?:^|\n)\s*([A-Z][a-z]{2,})\s+([A-Z][A-Za-z&\s\.]+?),?\s*(1[789]\d{2})`
- 1,170 raw genera, 181 family headers 추출
- 출력: `data/treatise_1959_genera_ocr.json`

### 3. Tree 통합 (`scripts/build_treatise1959_tree.py`)

- OCR author 교정 (~30개 매핑: "Howerr"→"HOWELL", "Kosayasut"→"KOBAYASHI" 등)
- 잘못된 항목 필터링 (family/subfamily 접미사, 비정상 연도 등)
- 페이지 위치 기반 genus→family 배정
- 최종: **1,159 unique genera** (1,048 tree에 배치)

### 4. OCR 이름 검증

DB의 5,089 genera와 대조:
- 정확 일치: 968 (83.3%)
- Fuzzy 교정 가능 (≥85%): 158
- 불확실 (75-85%): 26
- **완전 노이즈: 9개** (아래 참조)

### 5. Assertion DB Import (`scripts/import_treatise1959.py`)

재실행 가능한 import 스크립트 작성:

1. 기존 `treatise1959` 프로필 삭제 (edge cache, assertions, reference)
2. Fuzzy name cleaning (difflib ≥85% cutoff)
3. 수동 교정 (Ingieheidia→Inglefieldia, Gelemnurus→Illaenurus)
4. 노이즈 7건 제거
5. Reference 입력 (Moore 1959)
6. Tree 순회 → taxon match/create → PLACED_IN assertion 입력
7. Edge cache 구축 (default edges 복사 후 1959 edges로 교체)
